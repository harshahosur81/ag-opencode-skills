# SkillsMaster: Antigravity Skills Collection

Welcome to the **SkillsMaster** repository! This is a comprehensive collection of specialized "skills" designed to extend the capabilities of the Antigravity agent.

## Overview

Each skill in this repository is a self-contained folder containing instructions, scripts, and resources that provide Antigravity with expert-level knowledge and automation patterns for specific domainsâ€”ranging from backend architecture and security auditing to SEO optimization and startup metrics.

## Repository Structure

The core content is located in the `skills/` directory. Each skill follows a standard format:

- `SKILL.md`: The primary documentation and instruction set for the skill.
- `scripts/`: (Optional) Helper scripts and automation utilities.
- `examples/`: (Optional) Reference implementations and usage patterns.
- `resources/`: (Optional) Templates, assets, or data files.

3.  The agent will follow the documented guidelines and use any available scripts or resources to fulfill your request.
